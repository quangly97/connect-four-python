# Connect 4 Game

## Brief
A Connect 4 game implemented with python and its pygame library.

## Methods Used
* create_board()
* draw_board()
* column_available(board, col)
* get_open_row(board, col)
* drop_piece(board, row, col, piece)
* connect_four(board, piece)

## Technologies Used
* Python
* Git
* Github
* pygame
* numpy
